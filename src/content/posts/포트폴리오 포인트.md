#### 팀장 : 기술나열 => X

---

1. 실제로 개발 => 블로그 / 노션 / 본인이 공부한 git
2. 코드 이해 => 설명 : git / read me
3. 협업 => 문제점 (해결능력)
4. 개발자로 성장

---

1. 소개 : 본인

- 핵심 스택
- Git / 블로그 / 배포 URL(학원 => 동영상)

2. 프로젝트 소개
   => 산출물

- 주요기능
- 기술 스택
- 본인 역할
  - 문제 : 해결
  - 테스트 : 품질관리 => JUnit
  - CI/CD 배포 (캡처)

3. 학습

- 산출물

---

이력서

- 희망: 자바 웹개발 / SW 개발
- 기능:
  - backend => JPA
  - front => react
  - 기타 => docker, jenkins

프로젝트

- 카페 - 이력서
- 1/2차 포트폴리오 설명
- ppt 별첨

ppt (20~30page)

- 이름, 전화, 이메일(or 깃)
- 목차
  - 자기소개
  - 경력
  - 스킬
  - 프로젝트 목록
    - 프로젝트 1
    - 프로젝트 2
- 자기소개
- 기술 보유
  - 기술 이미지, 배운 내용?
- 프로젝트 목록
  - 짧은 표 (이력서에 있는거?)
- 프로젝트 1
  - 소개
  - 화면 UI
  - 사용기술
  - 프로젝트 성과/소감
  - 트러블 슈팅
  - CI/CD 캡쳐?
  - 추가 구현 사항 (예: 무중단)
- 프로젝트 2
