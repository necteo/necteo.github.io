### html

사실 이거 땜에 물어봄..

| 구분  | 추천 표기법           | 예시             | 이유                                    |
| :---- | :-------------------- | :--------------- | :-------------------------------------- |
| class | 케밥 케이스 (-)       | product-list     | CSS 스타일링 표준과 일치                |
| id    | 캐멜 케이스 또는 케밥 | userId / user-id | JS 변수와 매칭하거나 스타일과 통일      |
| name  | 스네이크 케이스 (\_)  | user_nameDB      | 컬럼명과 일치시켜서 서버 전송 시 편리함 |
