title: "giscus 추가"
---

giscus를 추가해보겠다

layout에 comments.html을 추가했다

```
---
layout: default
---

{% raw %} {{ content }} {{ page.comments.append }} {% endraw %}
```

원래 `layout`을 `post`로 했었는데

그랬더니 상위 레이아웃이 post가 되고

yml에서 기본 레이아웃을 post로 설정해둬서

적용이 안되는 상태였다

그래서 post.html의 layout을 comments로 바꾸고

comments.html이 layout을 default로 바꾸니 적용이 된다
