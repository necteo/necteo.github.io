### Vue

Main.vue

```js
<template>
<!-- 여기에 HTML 작성 (View) -->
</template>

<script>
/* Vue Script 작성 */
const app = Vue.createApp({
  data() {
    return {
      // Model
    }
  },
  mounted() {
    // 마운트 시 실행
  },
  methods: {
    // 사용자 정의 함수들 => ViewModel
  }
}).mount('#main')
</script>

<style scoped>
/* 해당 문서에서만 적용되는 CSS 설정 */
</style>
```

### Vue CDN

view.jsp

```jsp
<html>
<body>
  <div id="main">
    <!-- View -->
  </div>
  <script src="main.js"></script>
</body>
</html>
```

main.js

```js
/* Vue Script 작성 */
export default {
  data() {
    return {
      // Model
    };
  },
  mounted() {
    // 마운트 시 실행
  },
  methods: {
    // 사용자 정의 함수들 => ViewModel
  },
};
```
